# Trip Planner (Fractional Knapsack )

# Overview
Smart Trip Planner is an optimization-based project that helps users select the best travel destinations within a limited budget.  
It uses the **Fractional Knapsack Algorithm (Greedy Approach)** to maximize the total enjoyment value.

This project follows a **hybrid approach**:
- **C++** → for fast algorithm computation  
- **Python** → for user interaction and execution  

# Features
- Budget-based trip planning  
- Supports **partial selection** of destinations  
- Optimized using greedy algorithm (value/cost ratio)  
- Hybrid implementation (C++ + Python)  
- Clean and simple CLI interface  

# Algorithm Used
# Fractional Knapsack
- Items (destinations) can be selected **fully or partially**
- Selection is based on **maximum value per cost ratio**
- Ensures **optimal solution in minimum time**

# Tech Stack
- C++ (Core Algorithm)
- Python (Execution Layer)
- STL (Sorting, Vectors)
- Subprocess Module (Python ↔ C++ integration)
