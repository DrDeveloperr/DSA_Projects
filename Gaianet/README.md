# GaiaNet - AI for Regenerative Farming

# Overview
GaiaNet is an AI-based system that recommends suitable crops based on environmental conditions like temperature, humidity, and soil moisture.

# Features
-> Crop prediction using Machine Learning
-> Simple CLI interface
-> Real-world application in agriculture

# Tech Stack
-> Python
-> Machine Learning (Decision Tree)
-> Pandas, Scikit-learn

# How to Run
pip install -r requirements.txt  
python main.py
